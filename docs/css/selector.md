## CSS 權重分數

| 權重項目       | 權重比 |     備註 |
| -------------- | -----: | -------: |
| !importang     |  10000 | 最高權重 |
| 內聯樣式       |   1000 |          |
| id 選擇器      |   0100 |          |
| 類和偽類選擇器 |   0010 |          |
| 元素選擇器     |   0001 |          |
| 通用`*`選擇器  |   0000 |          |
| 繼承樣式       |        | 沒有權重 |

#### 備註

- 如果優先級計算後相同，此時則優先使用最下方的樣式。
- 當繼承遇到通用`*`選擇器，因本身就沒有權重，則會以通用`*`選擇器為優先判斷。

## 屬性選擇器

| 用法             | 說明                           | 範例            |
| ---------------- | ------------------------------ | --------------- |
| [屬性名]         | 選擇含有指定屬性的元素         | p[title]        |
| [屬性名=屬性值]  | 選擇含有指定屬性和屬性值的元素 | p[title='abc' ] |
| [屬性名^=屬性值] | 選擇屬性值以指定值開頭的元素   | p[title^='abc'] |
| [屬性名$=屬性值] | 選擇屬性值以指定值結尾的元素   | p[title$='abc'] |
| [屬性名*=屬性值] | 選擇屬性值中含有某值的元素     | p[title*='e']   |

## 偽類選擇器

### 種類一

> 這些偽類都是根據所有的子元素進行排序

| 用法         | 說明              | 範例                                                                                                             |
| ------------ | ----------------- | ---------------------------------------------------------------------------------------------------------------- |
| :first-child | 第一個子元素      |                                                                                                                  |
| :last-child  | 最後一個子元素    |                                                                                                                  |
| :nth-child() | 選中第 n 個子元素 | 特殊值:<br>n 第 n 個 n 的範圍 0 到正無窮<br>2n 或 even 表示選中偶數位的元素<br> 2n+1 或 odd 表示選中奇數位的元素 |

### 種類二

> 這幾個偽類的功能和上述的類似，不通點是他們是在`「同類型元素」`中進行排序

| 用法           | 說明 | 範例 |
| -------------- | ---- | ---- |
| :first-of-type |      |      |
| :last-of-type  |      |      |
| :nth-of-type() |      |      |

### 種類三

| 用法   | 說明                                     | 範例 |
| ------ | ---------------------------------------- | ---- |
| :not() | 否定偽類，將符合條件的元素從選擇器中去除 |      |

## 偽元素選擇器

偽元素使用 :: 開頭

| 用法           | 說明           | 範例                                          |
| -------------- | -------------- | --------------------------------------------- |
| ::first-letter | 表示第一個字母 |                                               |
| ::first-line   | 表示第一行     |                                               |
| ::selection    | 表示選中的內容 |                                               |
| ::before       | 元素的開始     | before 和 after 必須結合 `content` 屬性來使用 |
| ::after        | 元素的最後     |                                               |
